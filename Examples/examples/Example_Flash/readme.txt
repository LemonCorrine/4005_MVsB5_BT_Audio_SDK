Example_Flash工程主要演示了Flash的基本使用方法。

硬件环境要求:
    1. 使用BPxx系列开发板时，TX/RX/GND 接至 A9/A10/GND 引脚 (波特率 2000000)；

软件使用说明:
    上电启动之后，会打印如下信息，根据提示信息输入对应的字母，便可进入对应的示例程序。
    /-----------------------------------------------------\
    |                     I2S Example                     |
    |      Mountain View Silicon Technology Co.,Ltd.      |
    \-----------------------------------------------------/

	0 ---> erase
	1 ---> read
	2 ---> write
	3 ---> protect
	4 ---> unprotect
