ADC_Example工程主要演示了BP15系列SarADC采样相关示例。

硬件环境要求:
    1. BP15系列开发板
    2. 外接串口小板，TX/RX/GND 接至 A9/A10/GND 引脚 (波特率 256000)；

软件使用说明:
    1. 上电启动之后，通过串口接收指令，进入相应的电压采样case，并通过UART输出；
    2. 输入'a'，单次采样模式，采样GPIOA23端口相对值
    3. 输入'b'，单次采样模式，采样LDOIN的绝对电压，串口打印的是电压转换值，单位mV
    4. 输入'c'，单次采样模式，采样PowerKey端口的相对值
    4. 输入'd'，连续采样模式，DMA搬运采样数据，采样GPIOA28端口的相对电压

说明：
    1、注意需要选择参考电压源和模块工作时钟频率。
    2、注意GPIOA23，PowerKey在所在的Demo板或者开发板是否有相应的管脚，如果没有请更换测试管脚。
