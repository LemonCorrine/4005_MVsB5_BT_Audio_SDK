Example_PWC工程主要演示了PWC的基本使用方法。

硬件环境要求:
    1. 使用BPxx系列开发板时，TX/RX/GND 接至 A9/A10/GND 引脚 (波特率 2000000)；

软件使用说明:
    上电启动之后，会打印如下信息，根据提示信息输入对应的字母，便可进入对应的示例程序。
	-------------------------------------------------------
	PWC Example
	Example Menu:
	1: PWC Single mode example
	2: PWC Continue mode example
	3: PWC Interrupt example
	4: PWC DMA example
	-------------------------------------------------------
