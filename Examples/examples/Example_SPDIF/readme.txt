SPDIF_Example工程主要演示了SPDIF的基本使用方法。

功能：
SPDIF TX, SPDIF RX功能。

硬件环境要求:
    1. 使用BPxx系列开发板时，TX/RX/GND 接至配置好的引脚 (波特率默认2000000，可调整)；

软件使用说明:
    上电启动之后，会打印如下信息，根据提示信息输入对应的字母，便可进入对应的示例程序。

	/-----------------------------------------------------\
	|                     SPDIF Example                   |
	|      Mountain View Silicon Technology Co.,Ltd.      |
	\-----------------------------------------------------/
	please choose the example:
	
	R: SpdifRxExample
	T: SpdifTxExample
	A: ADCInSpdifTxExample
