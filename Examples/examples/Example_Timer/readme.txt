Example_Timer工程主要演示了Timer2的基本使用方法。

硬件环境要求:
    1. 使用BPxx系列开发板时，TX/RX/GND 接至配置好的引脚 (波特率默认2000000，可调整)；

软件使用说明:
    上电启动之后，会打印如下信息，根据提示信息输入对应的字母，便可进入对应的示例程序。
    -------------------------------------------------------
    Timer Example
    Examp Menu:
    a: Timer Interrupt mode example start,default is Timer2
    b: Timer Check mode example start,default is Timer2
    c: Timer Master Slave mode test
    n: Next Timer
    -------------------------------------------------------
